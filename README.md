# IT Marketplace – PHP Developers

A niche IT marketplace platform designed to connect **clients** with **PHP developers** through a structured, role-based system. The platform focuses on clear discovery, project posting, profile management, and efficient collaboration.

---

## Project Overview

This project solves the problem of **finding the right PHP developer quickly** by offering a dedicated marketplace instead of a generic freelance platform.
It supports **two user roles**:

* **Clients** – post projects and hire developers
* **Developers** – showcase skills and apply to projects

---

## Core Features

* Role-based authentication (Client / Developer)
* Dedicated Join Page with **Free, Pro, and Premium** plans
* Developer profile listings with search & filters
* Project posting and detailed project pages
* Dashboard for managing profiles, projects, and activity
* Secure login, signup, and password recovery
* Analytics for user engagement and activity tracking

## Navigation Flow

* Users land on **Home**
* Signup/Login based on role
* Redirected to **Dashboard**
* From Dashboard:

  * Manage Profile
  * Create or View Projects
  * View Search Results
  * Upgrade Plan (Join Page)

---

## Search & Discovery

* Search developers by skills, experience, budget, rating
* Filters connected directly to user profile data
* Premium profiles are visually highlighted

## Membership Plans
